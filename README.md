## How to run the project

- Have python3 installed
- Have pip installed
- Have swig ```sudo apt install swig```

Then run:
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

To run the programs:
```
./src/run.sh
```
